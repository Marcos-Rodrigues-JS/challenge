A API está hospedada na Vercel e foi desenvolvida para consultar o repositório da takenet (Blip) usando certos parâmetros. Seu principal objetivo é retornar os repositórios mais antigos. Em seguida, a API realiza um filtro para selecionar apenas os repositórios que utilizam a linguagem de programação C#.

No chatbot, implementamos um carrossel que interage com essa API. O carrossel exibe os 5 repositórios mais antigos, apresentados em ordem crescente, proporcionando aos usuários uma visão rápida e organizada dos projetos mais antigos desenvolvidos pela Blip na linguagem C#.
